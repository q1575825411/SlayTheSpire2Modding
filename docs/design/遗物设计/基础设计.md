# 遗物设计

当前仓库尚未沉淀具体遗物清单；这一阶段先明确技术落点，避免后续重复改入口。

## 当前技术方案

- 使用 `RitsuLibFramework.CreateContentPack(Entry.ModId)` 做显式注册。
- 当前角色体系下的遗物进入 `SoldierRelicPool`，作为 `士兵` 角色内容。
- 具体遗物统一继承 `Scripts/Relics/MyFirstRelic.cs`。
- 单个遗物补齐后，需要同时登记：
  - `Scripts/Relics/<遗物名>.cs`
  - `MyFirstStS2Mod/localization/zhs/relics.json`
  - `MyFirstStS2Mod/images/relics/<遗物名>.png`
  - `Entry.Init()` 里的 `.Relic<SoldierRelicPool, <遗物名>>()`

## 单个遗物最小模板

```csharp
public sealed class ExampleRelic : MyFirstRelic
{
    public override RelicRarity Rarity => RelicRarity.Common;
}
```
