【字段说明】

类型：
Attack = 攻击牌
Skill = 技能牌
Power = 能力牌

稀有度：
Common = 铜
Uncommon = 银
Rare = 金

额外标签：
Retain = 保留
Exhaust = 消耗
Ethereal = 虚无
Innate = 固有

原版关键词规则：
消耗、保留、永恒、无法打出、虚无、固有等，均视为游戏本体自带关键词。
实现时直接使用原版 CardKeyword，不单独注册自定义关键词。
卡牌描述中只写非关键词效果文本；关键词本身依赖卡牌关键词栏和游戏悬浮提示显示，不重复手写到描述正文里。

Tag 规则：
当前版本不单独设计自定义卡牌 Tag。
能用原版关键词表达的效果，优先使用原版关键词，不用 Tag 代替。
如后续确实需要接入原版 Tag，再单独在设计文档中补充并逐张标注。
