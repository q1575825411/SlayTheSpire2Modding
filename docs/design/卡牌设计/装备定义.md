# 装备系统规则文档

## 1. 字段说明

### 卡牌类型

```text
Attack = 攻击牌
Skill = 技能牌
Power = 能力牌
Equipment = 装备牌
```

### 装备类型

```text
Weapon = 武器牌
Armor = 防具牌
Mount = 坐骑牌
Treasure = 宝物牌
```

### 稀有度

```text
Common = 铜
Uncommon = 银
Rare = 金
```

### 额外标签

```text
Exhaust = 消耗
Retain = 保留
Ethereal = 虚无
Innate = 固有
```

---

# 2. 装备系统总定义

```text
【装备系统】
装备系统由装备牌和装备槽位组成。

装备牌是一类独立的卡牌类型，不属于攻击牌、技能牌或能力牌。

装备牌打出后不会进入弃牌堆，也不会进入能力区，而是进入对应的装备槽位，并持续生效。

装备槽位负责承载装备牌，并限制同类装备的唯一性。

装备被替换后，其效果立即失效。

当前版本补充约束：
装备牌仅限士兵使用。
非士兵角色即使获得装备牌，也不能打出并装备到槽位。
```

---

# 3. 装备槽位

```text
【装备槽位】
你初始拥有 4 个装备槽位：

武器槽
防具槽
坐骑槽
宝物槽

这 4 个装备槽位当前定义为士兵专属槽位，不是所有角色默认共享的公共槽位。

每个装备槽位同时只能存在 1 件装备。
你可以同时拥有 1 件武器、1 件防具、1 件坐骑、1 件宝物。

当你打出一张装备牌时：
如果对应槽位为空，则直接装备。
如果对应槽位已有装备，则替换原装备。
被替换的装备立即失效。

当你打出新的同槽位装备时，替换旧装备。
```

---

# 4. 装备牌

```text
【装备牌】
装备牌是一类独立的卡牌类型，不属于攻击牌、技能牌或能力牌。

装备牌打出后不会进入弃牌堆，也不会进入能力区，而是进入对应的装备槽位。

当前版本中，只有士兵可以打出并装备装备牌。

装备牌在装备槽位中持续生效。
装备被替换后，其效果立即失效。

装备牌不视为能力牌，不受能力牌相关效果影响，除非特殊说明。

装备牌不吃强化。
默认情况下，装备牌没有升级分支，不能通过常规强化获得数值或效果提升。

只有明确提到“装备牌”或对应装备类型的效果，才会影响装备牌。
```

# 5. 四种装备牌

## 5.1 武器牌

```text
【武器牌】
武器牌是一类装备牌。
武器牌打出后进入武器槽。

武器主要强化“杀”、攻击牌、伤害结算或进攻节奏。

你同时只能装备 1 件武器。
当你打出新的武器牌时，替换已有武器。
```

适合承载的效果方向：

```text
提高“杀”的伤害
增加“杀”的使用次数
改变“杀”的目标
让“杀”附加状态
让“杀”造成生命伤害后触发收益
让“杀”未造成生命伤害时获得补偿
强化攻击牌结算
改变进攻节奏
```

示例格式：

```text
青龙偃月刀
1费｜Equipment｜Uncommon｜Weapon
你的“杀”如果没有造成生命伤害，抽1张牌。
```

中文格式：

```text
青龙偃月刀
1费｜装备牌｜银｜武器
你的“杀”如果没有造成生命伤害，抽1张牌。
```

```text
【防具牌】
防具牌是一类装备牌。
防具牌打出后进入防具槽。

防具主要强化防御、格挡、减伤、闪或受到攻击时的反应。

你同时只能装备 1 件防具。
```

```text
受到攻击后获得资源
防止部分负面状态
提高回合间防御稳定性
```

示例格式：

```text
八卦阵
1费｜Equipment｜Uncommon｜Armor
每回合第一次受到攻击时，获得1张“闪”。
```

中文格式：

```text
八卦阵
1费｜装备牌｜银｜防具
每回合第一次受到攻击时，获得1张“闪”。
```

```text
【坐骑牌】
坐骑牌是一类装备牌。
坐骑牌打出后进入坐骑槽。

坐骑主要提供机动性、节奏、防御预备、敌人意图应对或回合转换收益。
```


示例格式：

```text
绝影
1费｜Equipment｜Uncommon｜Mount
每回合开始时，获得3点临时格挡。

```

中文格式：


# 6. 装备结算规则

## 6.1 打出装备牌

```text
当你打出一张装备牌时，按照以下流程结算：

1. 判断该装备牌的装备类型。
2. 找到对应装备槽位。
3. 如果该槽位为空，直接装备。
4. 如果该槽位已有装备，移除旧装备。
5. 将新装备放入该槽位。
6. 新装备开始持续生效。
```

---

## 6.2 替换装备

```text
当装备被替换时：

1. 原装备离开装备槽位。
2. 原装备效果立即失效。
3. 新装备进入装备槽位。
4. 新装备效果开始生效。
```

默认情况下：

```text
装备被替换时，不触发额外奖励。
装备被替换时，不进入弃牌堆。
装备被替换时，不进入消耗堆。
装备被替换时，不回到抽牌堆。
```

除非卡牌文本特殊说明。

---

## 6.3 重复装备同名装备

```text
如果你打出一张与当前槽位中相同名称的装备牌，仍然视为替换装备。

旧装备失效。
新装备进入槽位。
```

如果该装备具有“装备时”效果，则会重新触发。

示例：

```text
当前武器槽已有青龙偃月刀。
你再次打出青龙偃月刀。

结果：
旧青龙偃月刀被替换。
新青龙偃月刀进入武器槽。
如果青龙偃月刀有“装备时”效果，则重新触发。
```

---

## 6.4 装备效果持续条件

```text
装备效果只在该装备位于装备槽位中时生效。

当装备离开装备槽位后，其效果立即停止。
```

例如：

```text
贯石斧
1费｜Equipment｜Uncommon｜Weapon
你的“杀”造成的伤害 +3。
```

只有当贯石斧位于武器槽时，“杀”才获得 +3 伤害。
如果贯石斧被青龙偃月刀替换，则该伤害加成立即失效。

# 7. 装备牌与其他卡牌类型的关系

## 7.1 装备牌不是能力牌

```text
装备牌不属于 Power。
装备牌不视为能力牌。
装备牌不会进入能力区。
```

因此：

```text
影响能力牌的效果，不会影响装备牌。
```

例如：

```text
本回合你的下一张能力牌费用变为0。
```

这个效果不能作用于装备牌。

---

## 7.2 装备牌不是攻击牌或技能牌

```text
装备牌不属于 Attack。
装备牌不属于 Skill。
```

因此：

```text
影响攻击牌的效果，不会影响装备牌。
影响技能牌的效果，不会影响装备牌。
```

---

## 7.3 只有明确提到装备牌的效果才会影响装备牌

```text
只有明确写明“装备牌”“武器牌”“防具牌”“坐骑牌”或“宝物牌”的效果，才会影响装备牌。
```

例如：

```text
抽1张装备牌。
```

可以检索装备牌。

```text
本回合你的下一张装备牌费用变为0。
```

可以影响装备牌。

```text
从抽牌堆中选择1张武器牌加入手牌。
```

可以检索武器牌。

# 8. 装备槽位与扩展规则

## 8.1 第一版固定 4 槽位

```text
第一版装备系统中，装备槽位固定为 4 个：

武器槽
防具槽
坐骑槽
宝物槽
```

不建议第一版开放：

```text
额外武器槽
额外防具槽
额外坐骑槽
额外宝物槽
```

原因：

```text
多个同类装备同时存在会显著提高平衡成本。
尤其是多个武器共存，会导致“杀”体系过度膨胀。
```

---

## 8.2 后续扩展建议

后续如果需要扩展，可以优先考虑：

```text
额外宝物槽
临时装备槽
一次性装备
战斗开始自动装备
装备被替换时触发效果
```

不建议优先开放：

```text
双武器槽
双防具槽
无限装备槽
```

尤其是双武器槽，会让以下组合过强：

```text
诸葛连弩 + 贯石斧
诸葛连弩 + 青龙偃月刀
贯石斧 + 青龙偃月刀
```
